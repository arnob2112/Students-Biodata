Features

  • Everyone can open an account using a unique username and password.
  • Everyone can verify their email by clicking a link which will be sent in their email box.
  • Everyone can see and update their information.
  • A student can give a connection request to any teacher. If the teacher accepts that request, then there will be a connection between them.
  • A teacher also can give a connection request to any student. If the student accepts that request, then there will be a connection between them.
  • There is a webpage to show all connections, all students/teachers, and requests.
  • Any user can delete any connection with one click.
  • There is a Question and Answer panel where a student will ask questions and a teacher will answer them.
  • There is a notification panel where a student will easily find which of his questions has been answered as well as a teacher will easily find which of his answers should be answered again.
  • There is a Notice Board where only teachers can post a notice and everyone can see notices.
